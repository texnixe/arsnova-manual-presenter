Title: Present answers to lecture questions
----
URL-Key: present-answers-to-lecture-questions
----
Cat: PI
----
Steps:

		
1:
	Subheader: 
	Action: Click into the Session dropdown menu in the header bar and select a session from the list of available session. 
	Details: &rarr; The session is loaded and the available questions are listed in the teacher's pane on the tab **Questions for lecture".

2:
	Subheader: 
	Action: Click on one of the questions or on the tab **Answers**.
	Details: >
		The tab **Answers** is opened. By default, the answer statistics are not shown.
		
3:
	Subheader:  Answers of the first PI round.
	Action: To present the answers of the first PI round, click on **Show**.  
	Details: >	
		&rarr; The answer statistics for the current question are shown.
		
		
		
		**Notes**
		 	
		 				
		 				- Use the arrows to navigate between questions.
		 	
		 				- Clicking on the button  **Show** toggles between showing and hiding the answers.
		

4:
	Subheader: Menu **Show**
	Action: Use the menu **Show** to define how answers are shown.
	Details: >
		You can choose between the following options.
		
			
			- **Always show answers**: If activated, the answer statistics for all questions is shown automatically. If deactivated, you need to click on **Show** each time you open the answer tab of a question.
			
			
			- **Show correct answers**: If this option is enabled, correct answers are show in green, incorrect answers in gray.

			
			- **Before Discussion (1st)**: If this option is enabled, the answers of the first PI round are shown when the button **Show** is clicked.  If not, the answers of the first PI round are not shown when you click on **Show**.
			
			
			- **After Discussion (2nd)**: If this option is enabled, the answers of the second PI round are shown when the button **Show** is clicked.  If not, the answers of the second PI round are not shown when you click on **Show**.


5:
	Subheader: Menu **Release**
	Action: Use the menu **Release** to specify if questions and/or answer are released to the students. 
	Details: >
	
				
				- **Questions**: If enabled, the question is release, i.e. the students can view and answer the question.

				
				- **Answer statistics**: If enabled, the students can view the answer statistics.
	
	
				- **Correct Answers**: If enabled, the students can view the correct answers.


6:
	Subheader: Second PI Round
	Action: To start the second PI round, click on the button **1st**.
	Details:>
		 &rarr; A dialog opens.
		 
		 **Note**: As soon as you start the second PI round, the answers of the first PI round are locked irrevocably.
		
	
7:
	Subheader: 
	Action: Click in **Continue**.
	Details: > 
		&rarr; The second PI round is started.

8:
	Subheader: 
	Action: The second PI round is shown.
	Details: &rarr;  **Dieser Screenshot muss noch ausgetauscht oder bearbeitet werden oder weglassen und nur den n√§chsten.**

9:
	Subheader: 
	Action: To presend the answers, click on **Anzeigen**.
	Details: &rarr; The answers of both PI rounds are shown and can be compared.
